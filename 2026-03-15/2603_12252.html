1. Overview

Problem Addressed

Current MLLM-integrated diffusion frameworks for complex spatial/logical reasoning tasks suffer from two critical limitations:

  1. Insufficient MLLM reasoning depth: Single-step text encoding fails to activate the Chain-of-Thought (CoT) process required for accurate guidance on complex tasks.
  2. Static guidance during decoding: Fixed precomputed text embeddings prevent DiTs from progressively decomposing complex instructions into actionable denoising steps, even when MLLM encodings are correct.

Key Insight

Genuine CoT reasoning in diffusion models can be enabled by iteratively refining latent thought states in the MLLM to produce dynamic, evolving conditioning signals for the DiT, paired with terminal textual grounding to prevent reasoning drift. This endogenous (internally driven) reasoning paradigm outperforms static single-pass encoding approaches by 8.3 percentage points on average across complex reasoning benchmarks.

---

2. Key Contributions

  • First diffusion framework to enable genuine CoT reasoning via iterative latent state refinement, rather than relying on precomputed single-pass solutions.
  • Systematic empirical analysis including layer-wise sensitivity and attention entropy measurements that identify the two core bottlenecks limiting prior reasoning-enabled diffusion models.
  • State-of-the-art performance across 4 diverse reasoning benchmarks (Maze, TSP, VSP, Sudoku) with 25-40% accuracy gains on high-complexity task variants, plus controllable inference-time compute scaling and progressive image editing capabilities.

3. Method

The EndoCoT framework builds on flow matching as its base generative objective, with three core components: iterative thought guidance, terminal thought grounding, and a two-stage progressive training strategy.

Preliminary: Flow Matching

The standard flow matching objective defines a linear interpolation path between clean data and noise:

$$X_t = t X_1 + (1 - t) X_0$$

where $X_0 \sim \pi_0$ is a clean sample, $X_1 \sim \pi_1$ is Gaussian noise, and $t \in [0,1]$ is the flow timestep. The ground-truth vector field is:

$$u_t(X_t) = \frac{dX_t}{dt} = X_1 - X_0$$

The training objective for the DiT $v_\theta$ (conditioned on signal $c$) is:

$$\mathcal{L}_{\text{FM}} = \mathbb{E}_{t, X_0 \sim \pi_0, X_1 \sim \pi_1} \left\| v_\theta(X_t, t, c) - u_t(X_t) \right\|^2$$

These formulas form the base generative objective for all diffusion steps in the framework.

Iterative Thought Guidance

"Standard diffusion models generate an output by conditioning on static text embeddings. In contrast, our approach performs $\mathcal{T}$ reasoning steps, where each step refines both the visual output and the conditioning signal."

The reasoning process iteratively updates latent thought states in the MLLM $f_\phi$:

$$\textcolor{blue}{\mathbf{h}_\tau} = \mathbf{e}_{L+1}^\top f_\phi\left( [\mathbf{P} ; \textcolor{blue}{\mathbf{h}_{\tau-1}}] \right), \quad \tau = 1, \dots, \mathcal{T}$$

where $\mathbf{P}$ is fixed prefix embeddings from the input prompt and image, $[\cdot;\cdot]$ denotes sequence concatenation, and $\mathbf{e}_{L+1}$ extracts the $(L+1)$-th hidden state (the thought token) from the MLLM output. Each thought state $\mathbf{h}_\tau$ conditions the DiT to generate an intermediate visual output $\mathbf{I}_\tau$, with supervision against ground-truth intermediate targets (e.g., partial maze paths) via the reasoning loss:

$$\mathcal{L}_{\text{reasoning}} = \mathbb{E}_{\tau, t, \mathbf{z}_{\tau}(0), \mathbf{z}_{\tau}(1)} \left[ \| (\mathbf{z}_{\tau}(0) - \mathbf{z}_{\tau}(1)) - v_\psi(\mathbf{z}_\tau(t), t, \textcolor{blue}{\mathbf{h}_\tau}) \|^2 \right]$$

Terminal Thought Grounding

To prevent drift in latent reasoning states, the final thought state $\mathbf{h}_\mathcal{T}$ is aligned to a reference state derived from ground-truth textual reasoning steps:

$$\textcolor{blue}{\mathbf{h}_{\text{ref}}} = \mathbf{e}_{L_{\text{gt}}+1}^\top f_\phi\left( [\mathbf{P}_{\text{gt}} , \mathbf{I}_{\text{input}}] \right)$$

where $\mathbf{P}_{\text{gt}}$ is the embedding of the ground-truth reasoning trace. The alignment loss is:

$$\mathcal{L}_{\text{align}} = \| \textcolor{blue}{\mathbf{h}_\mathcal{T}} - \textcolor{blue}{\mathbf{h}_{\text{ref}}} \|^2$$

The total training loss combines the flow matching and alignment objectives:

$$\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{FM}} + \mathbb{I}_{\{\tau = \mathcal{T}\}} \cdot \lambda_{\text{align}} \mathcal{L}_{\text{align}}$$

where $\mathbb{I}_{\{\tau = \mathcal{T}\}}$ is an indicator function activating the alignment loss only at the final reasoning step, and $\lambda_{\text{align}}=1$ empirically.

Progressive Training Strategy

To avoid conflicting gradients between intermediate reasoning steps and final output quality, a two-stage training approach is used:

  1. Stage 1 (Reasoning Development): Supervise all $\mathcal{T}$ reasoning steps to learn step-by-step reasoning trajectories:

$$\mathcal{L}_{\text{stage1}} = \sum_{\tau=1}^{\mathcal{T}} \left( \mathcal{L}_{\text{FM}}^{\tau} + \mathbb{I}_{\{\tau = \mathcal{T}\}} \lambda_{\text{align}} \mathcal{L}_{\text{align}} \right)$$

  1. Stage 2 (Terminal Consolidation): Freeze gradients for intermediate steps, only optimizing the final output to improve quality while preserving learned reasoning dynamics:

$$\mathcal{L}_{\text{stage2}} = \mathcal{L}_{\text{FM}}^{\mathcal{T}} + \lambda_{\text{align}} \mathcal{L}_{\text{align}}$$

"intermediate steps $\tau < \mathcal{T}$ are executed in the forward pass only, serving as reasoning scaffolding without gradient propagation."

Inference

At inference time, the model recursively updates its latent thought states for $\mathcal{T}$ steps, with no requirement to decode intermediate visual outputs. The number of reasoning steps $\mathcal{T}$ can be adjusted to trade off compute cost and accuracy.

4. Experiments

Setup

EndoCoT is built on the Qwen-Image-Edit-2511 base model, with LoRA fine-tuning (rank 32, learning rate $1 \times 10^{-4}$, 5 epochs) applied to both the MLLM and DiT components. Training uses a 182.4K instance dataset spanning Maze, TSP, Sudoku, and VSP tasks. Evaluations are run under three settings: zero-shot, task-specific training, and unified training (single model trained on all tasks).

Key Results

  • Task-specific training: Average accuracy of 92.1% across all benchmarks, outperforming the strongest baseline (DiffThinker) by 8.3pp. On high-complexity variants: 90% accuracy on Maze-32 (vs 65% for DiffThinker), 95% on Sudoku-35 (vs 55% for DiffThinker).
  • Unified training: Average accuracy of 84.2% vs 77.1% for DiffThinker, demonstrating transferable reasoning skills across tasks.
  • Ablations: Removing the semantic alignment loss reduces Maze-32 accuracy from 90% to 14%; using explicit discrete reasoning tokens instead of latent states results in 0% accuracy on Maze-32 due to autoregressive error accumulation.
  • Inference scaling: Maze-32 accuracy increases linearly with the number of reasoning steps $\mathcal{T}$, from 11% at $\tau=2$ to 90% at $\tau=50$, with a predictable linear compute tradeoff.
  • Image editing: EndoCoT supports controllable progressive editing, with step-by-step object addition/transformation controlled by adjusting $\tau$.

6. Limitations & Future Work

Limitations

  1. Requires manual tuning of the optimal number of reasoning steps $\mathcal{T}$ for each task.
  2. Relies on high-quality datasets with explicit intermediate supervision signals for training.

Future Work

  1. Develop adaptive mechanisms to automatically adjust reasoning depth based on task complexity at inference time.
  2. Extend the framework to broader general-purpose tasks beyond specialized reasoning and image editing use cases.
