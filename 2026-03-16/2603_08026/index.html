Overview

Masked Diffusion Language Models (MDLMs) address the sequential generation bottleneck of Autoregressive Language Models (ARLMs) by enabling parallel token decoding via iterative denoising. However, their computational overhead is prohibitively high, as the bidirectional attention mechanism requires processing the entire input sequence at every denoising step, eroding the throughput gains of parallel decoding. The core insight of this work is that most token representations remain stable across consecutive diffusion steps, with only a small subset (termed salient tokens) undergoing meaningful changes that contribute to generation quality. The proposed DyLLM framework is a training-free inference optimization that exploits this temporal sparsity by selectively recomputing only salient tokens, reusing cached activations for non-salient tokens to minimize redundant computation.

Key Contributions

  • Layer-Adaptive Saliency Mechanism: A dynamic per-layer token selection policy that identifies salient tokens using cosine similarity of attention contexts between consecutive denoising steps, enabling redundant feed-forward network (FFN) computation to be skipped for stable hidden states.
  • Saliency-Aware Approximate Attention: A low-cost approximate attention mechanism that leverages activation sparsity to eliminate redundant context updates, reducing the native quadratic complexity of attention operations.
  • Scalable Throughput Improvements: A hardware-optimized, training-free framework that scales robustly with increasing parallel decoding degrees, delivering up to 7.6× higher throughput for LLaDA 8B and 9.6× higher throughput for Dream 7B across reasoning and code generation benchmarks, while largely preserving or improving baseline accuracy.

Saliency Detection

DyLLM quantifies representation stability using the temporal cosine similarity of attention contexts between consecutive diffusion steps, defined as:

$$s_{t,l}^{(i)} = \frac{C_{t,l}^{(i)} \cdot C_{t-1,l}^{(i)}}{\|C_{t,l}^{(i)}\| \|C_{t-1,l}^{(i)}\|}$$ where $C_{t,l}^{(i)}$ is the attention context vector for token $i$ at layer $l$ and timestep $t$. Tokens with $s_{t,l}^{(i)} \le \tau$ (user-specified threshold) are marked as salient, requiring full recomputation, while non-salient tokens reuse cached activations from the prior step. The distribution of cosine similarity values across layers confirms that most tokens have near-unity similarity across steps, validating the temporal sparsity assumption: > "DyLLM identifies salient tokens by measuring the cosine similarity of attention contexts between adjacent denoising steps. It recomputes feed-forward and attention operations only for salient tokens while reusing cached activations for the remainder." The validity of cosine similarity as a saliency metric is supported by two formal propositions: 1. Scale Invariance under Linear Projection: $\text{RMSNorm}((\alpha C) W_o) = \text{RMSNorm}(C W_o)$, where $\alpha$ is a positive scaling factor and $W_o$ is the attention output projection matrix. This confirms that only the direction of the attention context (measured by cosine similarity) affects the FFN input, not its magnitude. 2. Error Bound via Directional Alignment: $\delta \le \kappa(W_o)\sqrt{2(1 - s_{t,l})}$, where $\delta$ is the approximation error between normalized projected attention contexts of consecutive steps, and $\kappa(W_o)$ is the condition number of $W_o$. This bound quantifies the maximum error introduced by reusing non-salient token activations. > "When a token's representation is stationary $s_{t,l}^{(i)} \to 1$, skipping the FFN computation introduces almost no error. Conversely, tokens with low $s_{t,l}^{(i)}$ possess a large temporal shift, making them salient to the final generation quality." The number of salient tokens per layer follows a consistent trend across datasets, peaking at ~3/4 of the model depth before decreasing in final layers: ### Saliency-Aware Approximate Attention To reduce attention complexity, DyLLM decomposes the temporal change in attention context into content updates and attention weight re-routing: $$\begin{aligned} \Delta C_{t,l} &=(S_{t-1, l} + \Delta S)(V_{t-1,l}+\Delta V) - S_{t-1, l}V_{t-1, l} \\ &= (S_{t-1, l} + \Delta S)\Delta V + (\Delta S)V_{t-1,l} \end{aligned}$$

where $S_{t,l}$ is the attention score matrix and $V_{t,l}$ is the value matrix at layer $l$ and timestep $t$. This decomposition enables a dual-path attention implementation:

  1. Salient Path: Full exact attention computation is performed for salient tokens, capturing all changes to attention weights and values.
  2. Non-salient Path: Approximate column-sparse updates are applied, where non-salient tokens only receive updates from salient tokens, eliminating the need for full context recomputation.

This reduces attention complexity from $O(N^2d)$ to $O(N \cdot |\mathcal{A}_{t-1,l}|d)$, where $N$ is sequence length, $d$ is head dimension, and $|\mathcal{A}_{t-1,l}|$ is the number of salient tokens. Experiments

All experiments were run on a single NVIDIA H100 PCIe 80GB GPU, evaluating LLaDA 8B and Dream 7B instruct models against original implementations, Fast-dLLM (Prefix and Dual variants), and dLLM-Cache baselines across four benchmarks: GSM8K (math reasoning), MBPP (code generation), MATH (advanced mathematical reasoning), and MMLU-Pro (general knowledge).

Limitations & Future Work

Limitations

  1. The optimal cosine similarity threshold $\tau$ requires per-model and per-dataset tuning to balance speed and accuracy, with no universal default value across all use cases.
  2. Custom CUDA kernels for sparse attention operations limit portability to non-NVIDIA hardware platforms.
  3. Evaluations are limited to text generation MDLMs, with no testing on multi-modal or larger-scale diffusion models.

Future Work

  1. Develop automated threshold tuning mechanisms to enable zero-shot adaptation of DyLLM across tasks and models without manual configuration.
  2. Integrate DyLLM with confidence-aware decoding schemes to further increase parallel decoding degrees and throughput.
  3. Extend the saliency-based optimization framework to multi-modal diffusion models and other discrete diffusion generative tasks beyond text.
